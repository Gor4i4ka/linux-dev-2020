Оригинал "программы" лежит в src, на английском.
Перевод осуществлен на русский и po файл лежит в po/ru/task9.po.
Конфиг doxygen лежит в docs.
В libs лежат костыли.

1)Для сборки:
cd build
cmake ../
make

2) Языки
Русский язык программы запускается:
LANG=ru_RU.utf8 ./task9

Английский язык программы запускается:
LANG=en_US.utf8 ./task9

 (может разниться в различии от установленных диалекторв)
3) Опции
Добавлено 2 опции,
	-r для включения римских чисел
	--help для вывода помощи
(чтобы увидеть help другого языка надо на него переключиться)

4) Чистка делается:
make allclean

5) Изменение pot и po в случае изменения кода производится в build:
make update_po

(После изменения, чтобы создать mo надо еще раз сделать make)

6) Чтобы создать index.html, то есть заглавную страницу для doxygen, надо:
	Для русского help:
	make ru_help_intro
	Для английского help
	make en_help_intro
После этого надо еще раз сделать make, и теперь index.html обновился.

7) Чтобы установить man, нужно:
make
make install_man
(! man устанавливается в usr/locale/man/man3 и поэтому затребует пароль. Увы, лучше пока не сделал)
P.S. doxygen, оказывается, сам генерирует хороший man, лучше чем из help2man. Поэтому я использовал его.
