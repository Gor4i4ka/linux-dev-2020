CMake проект собирается в директории build, чтобы не мусорить.
Я пытался запустить CTest Coverage, но видимо я что-то не дописал а примера не было.
Посему я успел рефакторнуть 3 теста из всех, но, думаю, это не принципиально.
Пришлось создать примитивный хардкод анализатор для файла-лога.
Итак, порядок работы:

cd build
cmake ../
make
make test

Чтобы запустить примитив-проверку покрытия переходим в bin
Видим 3 теста и Coverage

запускаем Coverage с параметром-путем к файлу с логами от CTest

cd bin
./Coverage ../Testing/Temporary/*.log

3 из 3 тестов выполнились!

В итоге покрытие реализовано, но не так как надо и не над всеми тестами.
